# RC4 

## About

Rc4 is a binary additive stream cipher. It is used in SSL (also known as TLS), WEP and IEEE 802.11 wireless networking security standard

## Usage

```bash
# chmod +x rc4.py
```
For encryption
```bash
# ./rc4.py -e -m <message> -k <key>
```
For decryption
```bash
# ./rc4.py -d -m <message> -k <key>
```
